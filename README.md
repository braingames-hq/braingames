# Brain Games - Documentation Site

Static website for Brain Games iOS app. Hosts privacy policy, terms of service, and support pages.

## Pages

- `index.html` - Landing page
- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service
- `support.html` - Support / Contact

## Hosting

Deploy via GitHub Pages:

1. Go to repo Settings > Pages
2. Set source to "Deploy from a branch"
3. Select `main` branch, root `/`
4. Save

The site will be available at `https://<username>.github.io/braingames/`

## URLs for App Store Connect

Once deployed, use these URLs in App Store Connect:

- **Privacy Policy URL**: `https://<username>.github.io/braingames/privacy.html`
- **Support URL**: `https://<username>.github.io/braingames/support.html`
- **Marketing URL**: `https://<username>.github.io/braingames/`
